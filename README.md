# 🚚 Smart Delivery Route Optimization System

## 📌 Project Overview

Smart Delivery Route Optimization System helps users find the best route between Indian cities.

The system calculates:

- 📏 Distance
- ⏱ Travel Time
- ⛽ Fuel Cost
- 🗺 Route Visualization on Map

using OpenRouteService API and OpenStreetMap.

---

## ✨ Features

✅ Indian City Route Optimization

✅ Automatic Distance Calculation

✅ Travel Time Estimation

✅ Fuel Cost Prediction

✅ Interactive Route Map

✅ User Friendly Dashboard

✅ Flask Web Application

---

## 📊 System Workflow

1. User enters Source City
2. User enters Destination City
3. System fetches route information
4. Distance is calculated
5. Travel time is estimated
6. Fuel cost is predicted
7. Route is displayed on map

Source City: Tirupati

Destination City: Hyderabad

Distance: 556 KM
